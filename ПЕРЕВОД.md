## Языковые особенности

### Терминология
- **Company** → Компания (не "кампания", чтобы избежать путаницы)
- **Encounter** → Энкаунтер (устоявшийся термин в D&D сообществе)
- **Master** → Мастер (DM - Dungeon Master)
- **Player** → Игрок
- **Note** → Заметка
- **Privacy** → Приватность

### Стиль
- Используется вежливая форма обращения
- Короткие и понятные формулировки
- Консистентная терминология во всем приложении
- Сохранены англицизмы где это уместно (энкаунтер, дайс)

## Проверка

Для проверки перевода:

1. Откройте http://localhost:3001
2. Пройдитесь по всем страницам:
   - Главная
   - Регистрация/Вход
   - Компании
   - Энкаунтеры
   - Заметки
   - Персонажи
   - Броски кубиков
   - Справочник
   - Профиль

Весь текст должен быть на русском языке.
